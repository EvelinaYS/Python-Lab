# Отчет 

## Задание (Rare)
1. Создать репозиторий для дисциплины на GitHub.
2. Склонировать репозиторий на ПК.
3. Написать первую программу.
4. Запустить программу и убедиться в её работоспособности.
5. Выполнить коммит и пуш.
6. Оформить отчет в файле `README.md`.
7. Добавить шпаргалку в отчет по работе с `git`.

## Описание проделанной работы

В ходе выполнения практического задания был создан удаленный репозиторий на платформе GitHub. Репозиторий был склонирован на ПК. На языке Python была написана программа, выводящая в консоль фразу "Hello!". Программа была успешно запущена. После этого изменения были закоммичены и отправлены (pushed) в удаленный репозиторий на GitHub.
