## Задание 2. Круг

## 1. Описание задачи

Дано значение радиуса круга:

```python
radius = 42
```

Нужно вывести на консоль значение площади этого круга с точностю до 4-х знаков после запятой. Для этого необходимо использовать формулу площади круга: $$ S = \pi * R^2 $$
где $\pi$ = 3.1415926, $R = 42$, а округление выполняется с помощью функции `round()`.

Далее, даны координаты двух точек:

```python
point_1 = (23, 34)
point_2 = (30, 30)
```
Для каждой точки определить, лежит ли она внутри круга, и вывести на консоль `True` или `False`. Расстояние от точки до центра круга вычисляется по формуле: $$ d = √(x² + y²) $$
Точка лежит внутри круга, если d ≤ R.
